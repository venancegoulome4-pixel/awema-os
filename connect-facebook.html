Connecter Facebook & Instagram — guide pas à pas

Le plus puissant : un seul token Meta découvre toutes tes Pages et crée un client par Page, avec audience, réactions, commentaires à répondre, meilleurs posts… Instagram suit si un compte IG Pro est relié à la Page.

⏱️ ~10 min · 1 token → toutes les Pages

1 Être admin des Pages

Tu dois être administrateur des Pages Facebook de tes clients (ou recevoir un accès partenaire via leur Business Manager).

Ouvrir Business Manager ↗

★ Méthode simple — Se connecter avec Facebook sans terminal

Si tu as une app Facebook (developers.facebook.com → Créer une app, type Business), enregistre ses 2 clés ci-dessous, puis clique Se connecter : AWEMA récupère et allonge ton token tout seul (60 j).

2 Ouvrir l'explorateur d'API Graph

Connecte-toi avec le compte qui administre les Pages. C'est gratuit.

Graph API Explorer ↗
Si tu n'as pas d'app : il t'en propose une (« Meta App »). Crée-en une de type Business si besoin (developers.facebook.com → Créer une app).

3 Demander les bonnes permissions

Dans l'explorateur → Permissions, coche puis Generate Access Token :

pages_show_listpages_read_engagementpages_read_user_contentread_insights instagram_basicinstagram_manage_insights
Les 4 premières = Facebook ; les 2 dernières = Instagram (si comptes IG Pro reliés). Accepte la fenêtre d'autorisation Meta.

4 Allonger le token (60 jours)

Le token de l'explorateur est court (~1-2 h). Allonge-le en token longue durée (~60 j) via l'outil ci-dessous (Access Token Tool → Extend Access Token), puis copie le token long.

Access Token Tool ↗
Astuce pro : pour un token permanent, génère un System User token depuis le Business Manager (scopé aux Pages). Détails : guide Meta.

5 Donner le token à AWEMA & synchroniser sans terminal

Colle ton token long ci-dessous : AWEMA le range dans un coffre-fort privé de ton site (invisible au public), puis lance la découverte de toutes tes Pages (un client par Page).

⚠️ En mode dev (sans App Review), seuls les comptes testeurs sont lisibles. Pour gérer de vrais clients en production : Vérification d'entreprise + App Review. Voir le guide « Accès API (dev entreprise) » (docs/14).
Tu préfères la ligne de commande (Python en local) ?
python3 scripts/awema.py set meta META_TOKEN=TON_TOKEN
python3 scripts/connect-reseaux.py --meta-all && python3 outils/_data/build.py

7 Instagram (optionnel)

Pour les métriques IG : relie un compte Instagram Professionnel à la Page Facebook (dans les paramètres de la Page → comptes liés). Le même token le récupère ensuite automatiquement.
