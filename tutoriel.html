Installer AWEMA pour ton agence

Le guide complet, pensé pour débuter de zéro. Tu ne tapes aucune commande : tu cliques, colles, attends. Compte ~30 min la première fois.

🐝 Awa t'accompagne à chaque page · 100 % navigateur
L'idée en une image. AWEMA est un robot-assistant gratuit pour ton agence. Il a besoin d'un bureau pour vivre et ranger ses dossiers : ce bureau, c'est GitHub (gratuit). Ce guide, c'est : installer ton robot dans son bureau, lui donner une cervelle (une IA gratuite), et le brancher aux réseaux de tes clients pour qu'il aille chercher leurs vrais chiffres tout seul.

Le chemin (14 étapes)

  1. Créer un compte GitHub
  2. Copier le projet (fork)
  3. Allumer ton site
  4. La clé d'accès (token)
  5. Connecter GitHub à l'app
  6. Donner une cervelle (IA)
  7. Mettre ton agence à ta marque
  8. Ajouter tes clients
  9. Remplir leur ADN marketing
  10. Brancher les réseaux
  11. Lancer la récupération
  12. Lire ton cockpit
  13. Ça tourne tout seul
  14. Si ça coince

0 Créer un compte GitHub gratuit

GitHub, c'est le « bureau en ligne » gratuit où ton AWEMA va vivre. Si tu n'as pas encore de compte, crées-en un (un email + un mot de passe suffisent).

Créer mon compte GitHub ↗
👀 Tu sauras que c'est bon quand tu peux te connecter sur github.com et voir ton tableau de bord.

1 Copier le projet pour toi (le « fork »)

Un fork, c'est faire ta propre copie du projet AWEMA — comme photocopier un classeur modèle pour le remplir avec tes dossiers. Cette copie t'appartient, personne d'autre ne la voit.

Pourquoi : ainsi tes clients et tes chiffres restent chez toi, jamais mélangés avec ceux d'une autre agence.

🍴 Copier le projet (Fork)
👀 Tu sauras que c'est bon quand le projet apparaît sous ton nom : ton-pseudo / AwemA-….
⚠️ À ne pas sauter — le piège le plus courant. Sur une copie (fork), GitHub désactive l'automatisation par sécurité : sans ce clic, rien ne tournera tout seul (synchro, agents IA, publication), sans message d'erreur. Ouvre l'onglet Actions de ta copie ; si un bandeau jaune s'affiche, clique le bouton vert « I understand my workflows, go ahead and enable them ».
À refaire si tu laisses ta copie inactive plus de 60 jours (GitHub rendort les tâches planifiées).

2 Allumer ton site (« GitHub Pages »)

« Pages » transforme ta copie en un vrai site web — c'est l'écran d'AWEMA que tu vas utiliser tous les jours.

  1. Dans ta copie : clique l'onglet Settings (Réglages).
  2. Menu de gauche → Pages.
  3. Sous Source : choisis la branche main et le dossier / (root), puis Save.
  4. Patiente 1-2 min : GitHub affiche « Your site is live at … » avec ton adresse.
Ouvrir les réglages Pages ↗
👀 Note bien ton adresse https://ton-pseudo.github.io/AwemA-…/ — c'est l'adresse de ton AWEMA. Mets-la en favori et ouvre-la.
À partir d'ici, fais le reste depuis cette adresse (ton site), plus depuis github.com.

3 Fabriquer la « clé d'accès » (le token)

Pour qu'AWEMA puisse ranger des choses dans ton bureau GitHub tout seul, il lui faut une clé. On l'appelle un token (ou « PAT »). C'est une longue suite de caractères, à créer une seule fois.

  1. Ouvre la page de création (bouton ci-dessous).
  2. Token name : écris awema. Expiration : 90 jours (ou plus).
  3. Repository access → Only select repositories → choisis ta copie du projet.
  4. Permissions → Repository permissions → mets sur Read and write (lire+écrire) : Contents, Actions, Variables, Secrets.
  5. Tout en bas : Generate token → copie la suite de caractères (tu ne la reverras plus).
Créer ma clé (token) ↗
🔒 Cette clé est personnelle. Ne la partage avec personne et ne la colle nulle part en public. AWEMA la garde sur ton appareil, jamais dans le projet.

4 Présenter la clé à AWEMA

Ouvre la page Mise en route de ton AWEMA → bouton « 🔗 Connecter mon GitHub ». Une petite fenêtre s'ouvre : colle ta clé, ton pseudo (owner) et le nom de ta copie (repo). Valide.

⚙️ Ouvrir la Mise en route
👀 Tu sauras que c'est bon quand tu vois « ✅ Connecté » et que la pastille GitHub passe au vert. C'est l'étape clé : à partir de là, AWEMA fait tout le reste tout seul.

5 Donner une « cervelle » à ton robot (une IA gratuite)

Pour que les analyses et recommandations marchent, AWEMA a besoin d'une IA. Groq est gratuit et parfait pour démarrer. Crée un compte, génère une clé, copie-la.

Reviens sur la Mise en route → étape « Cerveau — une IA » → choisis Groq → colle ta clé → « ⚡ Enregistrer la clé IA ».

Obtenir une clé Groq ↗ Comparer les IA gratuites
👀 Tu sauras que c'est bon quand tu vois « ✅ branché », ou qu'une fenêtre t'ouvre le bon écran GitHub avec ta clé déjà copiée — il ne reste qu'à coller et valider.
Sans IA, AWEMA marche quand même : le feed « 3 choses à faire aujourd'hui » fonctionne. L'IA sert aux analyses et idées plus poussées.

6 Mettre AWEMA à ta marque

Donne à l'outil le nom, le logo et les couleurs de ton agence. Tout l'écran s'adapte ensuite.

🎨 Personnaliser mon agence
👀 Tu sauras que c'est bon quand le nom et les couleurs de ton agence apparaissent partout dans l'outil.

7 Ajouter tes clients

Crée une fiche par client. Le nom suffit pour commencer — aucun code, juste un formulaire.

➕ Ajouter un client
👀 Tu sauras que c'est bon quand ton client apparaît dans la liste et dans le cockpit.

8 Remplir l'« ADN marketing » de chaque client

C'est la mémoire du client : son ton, ses produits, sa cible, ses interdits. Plus elle est remplie, plus les idées et textes que l'IA propose sont justes.

🧠 Remplir la mémoire
👀 Bon réflexe : remplis-la au fil de l'eau. Même quelques lignes au début changent beaucoup la qualité.

10 Lancer la récupération des chiffres

Une fois un réseau branché, clique son bouton « Synchroniser » (ou « Découvrir mes Pages »). AWEMA va chercher les vraies données en coulisses.

👀 Patiente 1 à 2 minutes (le robot travaille), puis ouvre le cockpit : les chiffres réels apparaissent. C'est normal que ce ne soit pas instantané.

11 Piloter depuis ton cockpit

Le tableau de bord réunit tout : audience réelle, commentaires à répondre, meilleurs posts, cadence, et le feed « 3 choses à faire aujourd'hui ».

📊 Ouvrir le cockpit

12 Ensuite, ça tourne tout seul

Tu n'as plus rien à faire : AWEMA resynchronise périodiquement et fait travailler ses agents (analyste, stratège, créatif) chaque jour et après chaque synchro. Ton cockpit reste à jour sans intervention.

👀 Concrètement : tu ouvres ton AWEMA quand tu veux, les analyses et recommandations sont déjà fraîches.

13 Si quelque chose ne marche pas (c'est normal)

  • Message rouge ? C'est souvent une clé mal collée. Recommence l'étape — ça ne casse rien.
  • Pas sûr d'avoir réussi ? Cherche le ✅ vert ou le message « enregistré / lancé ».
  • Une synchro a échoué ? Sur GitHub → onglet Actions → clique le run rouge → regarde la ligne en erreur (souvent « secret manquant » = un token à reposer).
  • Perdu sur une page ? Clique Awa 🐝 en bas à droite : elle explique la page.
Bloqué ? Note à quelle étape et le message exact, et demande de l'aide — avec ça, on débloque vite.

🎉 Bravo, ton agence tourne sur AWEMA

Tu peux maintenant ajouter d'autres clients, brancher d'autres réseaux, et laisser les agents travailler pour toi.
