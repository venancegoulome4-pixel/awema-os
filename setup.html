5 · Mettre en ligne (auto-hébergement)

  1. Forke le projet sur ton compte GitHub (bouton « Fork » du dépôt).
  2. Place ton config/agence.json (téléchargé ci-dessus) à la racine du fork, dans config/, puis commit.
  3. Lance python3 outils/_data/build.py (ou le workflow le fait) pour régénérer config.js + le registre.
  4. Dans Settings → Pages du fork : source = branche main, dossier / (root). Le fichier .nojekyll est déjà présent.
  5. Ajoute tes accès dans Settings → Secrets and variables → Actions : META_TOKEN, YOUTUBE_API_KEY, TIKTOK_TOKENS (Variable), TIKTOK_PAT — selon les réseaux que tu connectes.
  6. Onboarde tes clients via Nouveau client et connecte leurs réseaux (guides TikTok · YouTube).
