Connecter LinkedIn — guide pas à pas

LinkedIn donne accès aux statistiques d'une Page entreprise (abonnés, impressions, engagement) via son API — après création d'une app et autorisation OAuth. C'est la plateforme la plus encadrée : ce guide te mène jusqu'au jeton.

⏱️ ~15 min · nécessite une Page entreprise dont tu es admin

1 Avoir une Page entreprise (admin)

L'API d'analytics LinkedIn fonctionne sur les Pages entreprise (pas les profils perso). Tu dois en être administrateur.

  • Page existante : note son URL, ex. linkedin.com/company/mon-entreprise.
  • Pas encore de page ? Crée-la (gratuit) via le bouton ci-dessous.
Créer une Page entreprise ↗

2 Créer une app LinkedIn Developer

  • Ouvre le portail développeur (bouton ci-dessous) → Create app.
  • Renseigne le nom (AWEMA), associe la Page entreprise de l'étape 1, accepte les conditions.
Créer l'app LinkedIn ↗

3 Vérifier la Page & demander les produits

  • Onglet Settings → Verify : génère le lien, ouvre-le en tant qu'admin de la Page pour vérifier l'app.
  • Onglet Products → clique « Request access » sur :
    • Sign In with LinkedIn using OpenID Connect (Standard Tier) — accordé immédiatement (le login OAuth).
    • Community Management API (Development Tier) — les stats de Page (abonnés, impressions, engagement).
✅ Le Development Tier de la Community Management API te donne accès tout de suite à tes propres Pages (celles dont tu es admin + app vérifiée) — pas besoin d'attendre une revue pour un usage AWEMA auto-hébergé. La validation complète ne sert que pour gérer des Pages tierces à grande échelle.
Ignore les autres produits (Advertising, Lead Sync, Conversions, Ad Library…) — inutiles ici.

4 Enregistrer Client ID & Client Secret sans terminal

  • Onglet Auth → copie le Client ID et le Client Secret.
  • Ajoute cette URL de redirection OAuth dans l'app : …

Colle-les ici : AWEMA les range en Secrets GitHub (jamais dans le dépôt).

Le PAT (Variables R/W) sert à enregistrer le token obtenu. Si tu l'as déjà posé pour TikTok (TIKTOK_PAT), inutile d'en remettre un — sinon enregistre-le ci-dessus en AWEMA_PAT.

5 Connecter la Page LinkedIn

✅ Tu cliques, AWEMA fait le reste en arrière-plan, sans rien te demander de plus. (En coulisses : il transforme tes clés en un jeton de lecture — tu n'as pas à t'en occuper.)
  • Scopes : r_organization_social, rw_organization_admin (analytics de Page).
  • Tu autorises sur LinkedIn → retour automatique sur AWEMA, qui finalise et enregistre LINKEDIN_TOKEN.

6 Synchroniser

Une fois la Page connectée, lance la synchro : abonnés, impressions et engagement arrivent dans la fiche, à côté de Facebook, TikTok et YouTube.

Ouvrir « Sync présence digitale » ↗

python3 scripts/linkedin-onboard.py (capture auto) ou python3 scripts/linkedin-onboard.py --paste (via oauth.html). L'access token affiché va dans le Secret LINKEDIN_TOKEN.
