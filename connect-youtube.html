Connecter YouTube — guide pas à pas

Le plus simple des réseaux : on lit les statistiques publiques d'une chaîne (abonnés, vues, vidéos) avec une clé API. Pas d'OAuth, pas de connexion par compte. Une seule clé sert pour toutes les chaînes.

⏱️ ~5 min · gratuit (quota large)

1 Ouvrir Google Cloud Console

Connecte-toi avec un compte Google. C'est gratuit.

Ouvrir la console ↗

2 Créer (ou choisir) un projet

  • En haut, clique le sélecteur de projet → Nouveau projet.
  • Nom : AWEMA → Créer. Sélectionne-le ensuite.

3 Activer « YouTube Data API v3 »

Bouton ci-dessous → page de l'API → Activer.

Activer YouTube Data API v3 ↗

4 Créer la clé API

  • Va dans Identifiants (bouton ci-dessous).
  • + Créer des identifiants → Clé API.
  • Copie la clé (elle commence souvent par AIza…).
  • (Recommandé) clique la clé → Restrictions d'API → limite à YouTube Data API v3 → Enregistrer.
Ouvrir « Identifiants » ↗
🔒 AWEMA range ta clé dans un coffre-fort privé de ton site : elle reste invisible et n'apparaît jamais sur tes pages publiques.

5 Repérer chaque chaîne à suivre

Pas besoin de chercher l'ID technique : donne-moi simplement le @handle ou l'URL de la chaîne — je résous l'ID automatiquement dans le connecteur.

  • Sur la chaîne → son adresse ressemble à youtube.com/@CodeScooper → le @CodeScooper suffit.
  • Ou colle l'URL complète de la chaîne.
L'ID/handle de chaque entité sera rangé dans son client.json (yt_handle), comme le fb_page_id.

6 Me transmettre 2 choses

  • La clé API (colle-la dans le champ ci-dessus — AWEMA la range en sécurité).
  • Le @handle / l'URL de chaque chaîne (Code Scooper, etc.) — ça, tu peux me le donner.

Le connecteur --youtube-all récupère abonnés, vues totales, nombre de vidéos + top vidéos, fusionnés dans la fiche de l'entité (à côté de Facebook & TikTok).
